命令：

```bash
adb shell dumpsys activity service com.android.car inject-vhal-event 0x26400241 0x10 1
```

参数说明：

- `0x26400241`：VHAL 属性 ID。从 ID 格式看，属于 OEM 自定义范围（标准 Android 属性通常在 `0x01000000`-`0x06000000`）。具体含义取决于车型定义，可能表示某个车门、车窗或座舱功能。
- `0x10`：Area ID，表示操作的区域。常见值：
  - `0x01`：前排左侧
  - `0x02`：前排右侧
  - `0x10`：可能表示全车或特定区域
- `1`：注入的值，通常是布尔值（0 = 关闭/false，1 = 开启/true）或状态值

## 前置条件

1. UserDebug 或 Eng 版本
   ```bash
   adb shell getprop ro.build.type
   ```
