# wsl2 ubuntu
`sudo vim /etc/resolv.conf`
添加以下内容：
```
nameserver 8.8.8.8
nameserver 114.114.114.114
```
> **NOTE:** 8.8.8.8 和 114.114.114.114 是常见的公共 DNS 服务器

可选：
`sudo vim /etc/wsl.conf`
添加：
```
[network]
generateResolvConf = false
```
>Note: 实测没什么作用，每次重新开机之后还是需要修改 /etc/resolv.conf
