使用rm删除文件的时候有时候会误删，且无法恢复，尝试给使用mv替代rm，但有时候还是会有一些问题。ubuntu上有直接可用的安全删除文件的工具trash-cli  

# install
sudo apt install trash-cli

# 常用命令
- 删除文件
    - trash filename
    - trash foldername/
- 查看回收站
    - trash-list

- 恢复
    - trash-restore
- 情况回收站
    - trash-empty
    - trash-empty 7 清除超过7天前的文件

> 也可以使用这个代替rm，给rm起个别名
```
echo "alias rm='trash'" >> ~/.bashrc
source ~/.bashrc
```
