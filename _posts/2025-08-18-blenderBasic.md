# 修改物体本地坐标系原点
1. 进入编辑模式，选中要作为新的原点的点
2. 对齐游标，按 Shift + S → Cursor to Selected
3. 返回物体模式，设置原点：右键 → Set Origin → Origin to 3D Cursor

# 测量
1. 按住ctrl吸附到点或者边
2. 按住xyz固定坐标轴
