# install
```
sudo apt-get install valgrind
``` 

# basic usage
```
valgrind --leak-check=full ./your_program
```

```
valgrind --leak-check=full \
         --show-leak-kinds=all \
         --track-origins=yes \
         --verbose \
         --log-file=valgrind-out.txt \
         bazel-bin/test/ceer/svm_test
```
